# 快速配置

> ChoeskiTool 是一个为**Choseki**开发的打包音源包的辅助工具
>
> 使用方法简单,使用需要安装Java Runtime Environment 1.8+

---

## 使用方法

``` bat
    java -jar "ChosekiTool.jar"
```
