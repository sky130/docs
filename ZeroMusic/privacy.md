# 零度音乐隐私政策

生效日期：2026年5月22日
最新修改日期：2026年5月22日

零度音乐由珠海市晞云云科技有限公司（以下简称“晞云公司”或“我们”）提供。我们重视您的个人信息保护。本政策说明我们在您使用零度音乐应用及相关服务时，如何收集、使用、存储、共享和保护您的个人信息，以及您如何管理相关权利。

请您在使用本应用前仔细阅读并理解本政策。您点击同意、继续使用本应用，或使用需要处理个人信息的功能，即表示您已阅读并同意本政策。若您不同意本政策，请停止使用本应用。

## 一、我们如何处理个人信息

为提供本地音乐播放、应用设置、Pro 购买与许可证校验等功能，我们可能在必要范围内处理以下信息：

1. 本地音乐和媒体信息：您设备中的音乐文件路径、文件名、音频时长、专辑、歌手、封面、歌词、播放记录、歌单、黑名单目录、自定义扫描目录等。该类信息主要存储在您的设备本地，用于扫描音乐、播放音乐、管理歌单和展示媒体信息。
2. 设备与应用信息：设备型号、系统版本、应用版本、网络状态、权限状态、运行状态、错误信息等，用于适配设备、排查故障、保障应用运行和提供售后支持。
3. Pro 许可证信息：设备标识哈希、订单号、激活方式、许可证状态、令牌有效期、剩余迁移次数、迁移二维码相关信息等，用于购买、兑换、激活、刷新、迁移和售后核验 Pro 服务。
4. 支付和兑换信息：支付订单号、支付状态、支付方式、支付时间、礼品码及兑换结果等，用于完成交易、查询订单、处理售后和防止欺诈。我们不会主动收集您的银行卡号、微信支付账号密码等支付账户敏感凭证。
5. 联系与售后信息：当您通过邮件或其他方式联系我们时，您主动提供的邮箱地址、问题描述、截图、订单信息、设备信息等，用于处理咨询、投诉、售后和争议。

## 二、权限使用说明

本应用可能申请以下系统权限。您可以在系统设置中管理授权：

1. 文件、媒体或所有文件访问权限：用于扫描、读取和播放音乐文件，读取封面、歌词和您主动选择的文件。
2. 网络权限和网络状态权限：用于检查更新、加载必要公告、Pro 购买、礼品码兑换、许可证校验和刷新。
3. 蓝牙权限：用于识别蓝牙设备连接状态、支持蓝牙歌词和相关播放体验。
4. 通知和前台服务权限：用于在后台持续播放音乐、展示播放控制和必要的运行状态。
5. 安装应用权限：仅在您主动下载并安装更新包时，用于调用系统安装流程。
6. 唤醒锁等系统能力：用于在播放过程中维持必要的播放服务运行。

关闭某些权限可能导致对应功能无法使用。例如，关闭文件读取权限后，本应用可能无法扫描或播放本地音乐；关闭网络权限后，可能无法完成 Pro 购买、兑换或许可证刷新。

## 三、我们如何使用信息

我们会将个人信息用于以下目的：

1. 提供、维护和优化本地音乐播放、媒体库扫描、歌单、歌词、封面、蓝牙歌词、文件传输、自定义设置等功能。
2. 完成 Pro 服务的购买、兑换、激活、设备绑定、本地验证、许可证刷新、设备迁移和售后处理。
3. 保障应用、支付和许可证系统安全，识别异常请求、伪造令牌、盗用礼品码、破解或滥用行为。
4. 响应您的咨询、投诉、错误反馈、退款或迁移协助请求。
5. 遵守法律法规、监管要求、司法或行政机关依法提出的要求。

我们不会将您的个人信息用于与上述目的无关的行为。如需将信息用于本政策未载明的其他目的，我们会依法另行告知并取得您的同意。

## 四、信息存储与保护

1. 本地音乐文件、歌单、播放记录、设置、缓存和自定义内容主要存储在您的设备本地。
2. 为提供 Pro 服务，我们可能在服务器保存订单号、设备标识哈希、激活方式、许可证状态、令牌有效期、迁移次数、兑换记录等必要信息。
3. 我们将在实现处理目的所必需的最短期限内保存您的个人信息。法律法规要求保存更长时间的，我们将依法保存。
4. 我们会采取合理的技术和管理措施保护个人信息安全，包括访问控制、加密或哈希处理、日志管理和必要的安全审计。但请您理解，任何网络传输和电子存储方式都无法保证绝对安全。

## 五、共享、转让与公开披露

1. 我们不会出售您的个人信息。
2. 为完成支付、订单查询、系统能力调用或法律要求，我们可能在必要范围内向支付服务提供方、系统服务、监管机关、司法机关或依法有权的机构提供必要信息。
3. 除以下情形外，我们不会向第三方共享、转让或公开披露您的个人信息：
   - 已取得您的单独同意。
   - 为履行法律法规规定的义务所必需。
   - 为订立、履行您作为一方当事人的合同所必需。
   - 为处理投诉、售后、争议或保护您、我们或他人的合法权益所必需，且在合理必要范围内。
   - 因合并、分立、解散、被宣告破产等原因需要转移个人信息的，我们会告知接收方名称和联系方式，并要求接收方继续受本政策约束；接收方变更原处理目的或方式的，将重新取得您的同意。

## 六、您的权利

您依法享有查阅、复制、更正、补充、删除个人信息，撤回同意，注销或停止使用相关服务，解释说明个人信息处理规则等权利。

您可以通过以下方式管理信息：

1. 在系统设置中关闭或管理权限。
2. 在应用内删除或修改歌单、扫描目录、黑名单、播放记录、缓存和相关设置。
3. 对 Pro 订单、许可证、设备迁移、退款或个人信息处理提出疑问时，通过客服邮箱联系我们。

请注意，撤回同意或关闭权限可能导致相关功能无法继续使用。为保障安全和核验身份，我们处理您的权利请求时可能需要您提供必要的订单号、设备信息或其他证明材料。

## 七、未成年人保护

若您未满18周岁，应在监护人的陪同和指导下阅读本政策，并在取得监护人同意后使用本应用。监护人如发现未成年人未经同意向我们提供个人信息，或希望行使查阅、更正、删除等权利，可通过本政策载明的联系方式联系我们。

## 八、政策更新

我们可能根据法律法规变化、产品功能调整或个人信息处理情况变化更新本政策。更新后的政策将在应用内、官网或其他合理渠道公布，并自公布之日起生效，法律法规另有规定或更新说明另有约定的除外。

如您不同意更新后的政策，请停止使用相关功能或本应用。您继续使用本应用，即视为您已阅读并接受更新后的政策。
