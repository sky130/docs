# 快速配置

> 一个可以获取小米手环的AuthKey的类库

---

## 在build.gradle中引进
``` java
  ddependencies {
    ...
    implementation('ml.sky233.suiteki:Suiteki:1.0.5')   
    implementation("com.squareup.okhttp3:okhttp:4.9.1")
}
```

## 在Java文件中引进
``` java
    ...
    import static ml.sky233.hitokoto.Hitokoto.*;
    import ml.sky233.hitokoto.HitokotoUtils;
    ...
```
